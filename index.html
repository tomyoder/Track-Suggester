The BIG question: Design or Function?

The first question is whether you're more interested in working with design or structure. Would you rather craft how a website looks and "feels" to use, or make the nuts-and bolts structure that makes the whole site function?

What kind of company would you rather work for?

Many programming students aspire to work for a large company, in part because of a perceived status, stability, and an expectation of high pay. Working for a large company can be very rewarding, though one can be "lost in the crowd". Large employers hiring programmers also include Government and Utilities.

C#

C# is in heavy use at Microsoft, and many other major software employers. C# is intended to be a modern object-oriented programming language. C# is used for applications in both hosted and embedded systems, making it useful pretty much anywhere, but most likely in apps intended for production on a mass scale.

Java

The programming language Java, originally designed for interactive television, has become one of the most widely used languages in the field. Java aspires to an idea know as WORA, or "write once, run anywhere". WORA means Java code can run on all platforms that support Java without the need for recompilation. Java is in use by large firms but is also renowned for its flexibility, open-sourced accessibility, and the fact that it is not certified to an ECMA or ISO/IEC standard but is subject to Java Community Process.

Medium sized companies offer much of the benefits of larger ones while being generally more flexible and dynamic. More recognition is available to individual employees, and strict adherence to specialization is lessened. Language needs begin to emphasize that same flexibility and adaptibility. Speed in prototyping apps is more of a priority than focusing on the requirements of mass production.

PHP

PHP is a server-side scripting language intended for web development. It's been said that 80% of the web is running on PHP. Like Java, PHP has no formal specification. PHP is most useful to websites dependent on access to databases in order to function, and not quite as dynamic in user interaction as JavaScript, the most comparable browser-side scripting language. PHP is also used as a general-purpose programming language.

Small companies can adapt more quickly to changing conditions than large ones, and the information industry changes faster than any other. Small companies offer individuals a real ability to point to a product and say "I built that."
